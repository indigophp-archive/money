**Value Object implementation of Fowler's Money pattern.**


## Install

Via Composer

``` json
{
    "require": {
        "indigophp/money": "@stable"
    }
}
```


## Usage


### Class Inheritance

Value Objects in general should be treated as `final`. This means instead of extending a class you should use decorators.
